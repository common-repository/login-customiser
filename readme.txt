== Description ==

A Simple plugin to customise WP-Login, allowing you to change where users are redirected to upon successful login.

== Installation ==

1. Install Login Customiser either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin.
3. Enable the updates that you would like to be applied automatically via the options menu.
== Frequently asked questions ==

= Will this delete any customisations I have made myself? =

If you have made some customisations to a plugin or theme, then simply exclude it from the update list (found in the settings page)

= How often will this check for updates? =

Currently the update is scheduled to run every hour.
